Jogo dos Ponteiros

Um jogo simples em plano cartesiano, feito para implementar IAs

  • Manual
  • Aleatório

Manual

O objetivo do jogo é fazer todos os ponteiros apontarem para cima. Quando a fase se inicia, eles irão apontar aleatoriamente entre Norte, Sul, Leste e Oeste. Você pode clicar com o botão esquerdo do mouse para girar no sentido horário, ou o botão do meio para girar no sentido anti horário

Simples!

Faça todos os ponteiros apontarem para o Norte

Gire!

Faça os ponteiros girarem com o botão do mouse

Aleatório

Uma IA tentará girar os ponteiros de forma aleatória e navegar pelo tabuleiro. A cada rodada ela poderá apenas fazer uma ação, consumindo um ponto

Navegação

Ir para esquerda, direita, cima ou baixo

Giro

Girar no sentido horário ou anti horário
